# Intro Econometrics Portfolio

This repository contains my applied econometrics work using R, based on exercises from *Introductory Econometrics: A Modern Approach* by Wooldridge.

## 📊 Overview
The goal of this portfolio is to demonstrate my understanding of core econometric concepts through practical implementation and interpretation.

## 🧠 Topics Covered
- Simple linear regression
- Interpretation of coefficients
- Statistical significance
- Model fit (R-squared)
- Economic interpretation of results

## 📁 Projects
Each folder contains:
- R scripts (`.R`) with analysis
- Notes (`.md`) explaining results and interpretation

### Included datasets:
- 401k participation
- CEO salary data
- Charity donations
- County-level crime data
- Education performance (MEAP)
- Sleep and productivity
- Wage determination

## 🛠️ Tools
- R
- RMarkdown

## 🚀 Purpose
This repository is part of my learning journey in econometrics and data analysis, with a focus on applying theory to real-world datasets.
